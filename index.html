¿Qué es el Frontend?

El Frontend o interfaz de usuario se dedica a optimizar la apariencia y experiencia de usuario, su objetivo es que el sitio sea facil de entender y usar

¿Qué es el Backend?

El Backend o interfaz de programacion es la parte que logra que el sitio web sea funcional y logico, este busca que el sitio responda a las instrucciones del usuario
